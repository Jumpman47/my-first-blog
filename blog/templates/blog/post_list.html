Senior Capstone Project

For my capstone project, I am tasked with building a webserver on a Raspberry pi 3 B+. We are using this webserver to connect to a camera server which is also running on a Raspberry Pi 3 B+. The camera server is running the motion software which records any video taken from a usb webcamera. My team and I will be using the webcam to monitor robots setup in our "Robotic Park". The webserver will also provide a means for users to interact with the robots and even perform basic commands so they can move the robots around the testbed.
